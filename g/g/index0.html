info What is Antigravity?

Google Antigravity is a standalone desktop IDE (Integrated Development Environment) released in November 2025. While it looks like VS Code (and is likely a fork), its core engine is fundamentally different. Instead of a simple "chat sidebar," it uses Autonomous Agents powered by Gemini 3 Pro.

Unlike previous AI tools that just suggest code, Antigravity agents have direct access to:

  • The Editor: To read, write, and refactor files.
  • The Terminal: To run builds, tests, and git commands.
  • The Browser: To visually verify web apps using a built-in Chrome engine.

Where are my files? Understanding the Environment

folder_open

Local File Storage

Just like VS Code, Antigravity works on your local Windows file system. Your project files live in standard folders (e.g., C:\Users\You\Projects). Agents edit these files directly.

inventory_2

Artifacts & Mission Logs

Agents generate "Artifacts" (plans, screenshots, test logs). These are often stored in a hidden .antigravity directory in your project root, but are best viewed through the Mission Control interface.

group_work

Multi-Agent Workspace

Agents don't just "talk". You can spawn separate agents for different tasks (e.g., one fixing a backend bug, one updating CSS) simultaneously. They share the file system but maintain separate context threads.

Installation Guide for Windows

1

Download the Installer

Visit antigravity.google/download (Simulated link) and select the Windows (x64) installer. The file is approximately 450MB.

2

Install & Authorize

Run the installer. On first launch, Antigravity will ask to sign in to your Google Account to access the Gemini 3 Pro API.
*Note: You do not need a paid subscription for the Public Preview.

3

Import or Create

Since it's based on VS Code, you can simply File > Open Folder to load your existing projects. Extensions from the VS Code marketplace are compatible!

Pro Tips & Insights

radio_button_unchecked

Try "Vibe Coding"

Don't worry about syntax. Just describe the "vibe" or functionality you want (e.g., "Make it look like a 90s hacker movie") and let the agent handle the CSS/HTML structure.

speed

Rate Limits

The preview has generous limits that refresh every 5 hours. Use the "Manager View" to track your token usage per agent.

extension

Browser Extension

Antigravity installs a local Chrome extension automatically. This is how agents "see" your web app to verify fixes. Do not disable it!
