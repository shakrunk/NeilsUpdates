The "Magic" isn't that the AI got smarter. It's that the constraints were tighter. By explicitly banning placeholders and demanding specific interactivity, you force the AI to "think" harder to fill the gaps, resulting in a richer application.

Act as an Expert Front-End Developer. Your goal is to build a polished, functional, and visually stunning web application.

TASK: Create a A data visualization tool for an organ donation organization that wants their staff to have both organization-wide and individual-technition-specific insides to boost growth.

REQUIREMENTS:
- SINGLE FILE: Output the complete application in a single HTML file containing all CSS (Tailwind via CDN) and JavaScript.

- INTERACTIVITY: The application must be functional. Users should be able to filter data, switch tabs, or toggle views using JavaScript.
- VISUALS: Use Google Material Symbols for icons. For images, use relevant Unsplash source URLs or styled placeholders (e.g., for avatars).
- DESIGN: Use a modern, professional color palette with Tailwind CSS. Ensure responsive design for mobile and desktop.

Please think step-by-step about the structure before generating the code.
